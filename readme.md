## L'api
___

J'utilise postgreSQL pour la base de données en localhost
<br>
### Faites ces commandes dans l'ordre pour que le front se lance
````cd api````

````npm install````

````node app.js````

## Le front
___ 
J'ai fait du vuejs 3 avec Pinia pour gérer les films et axios pour les requêtes.
<br>
### Faites ces commandes dans l'ordre pour que le front se lance

```cd front```

```npm install```

```npm run dev```
